## 🎯 Descrição do Desafio – Uso de Parâmetros no Power BI

Neste desafio, o objetivo foi aprimorar o relatório desenvolvido, incorporando o uso de **parâmetros de campos** para criar visões dinâmicas e interativas. A proposta é permitir que o usuário explore os dados sob diferentes perspectivas, mantendo consistência visual e construindo uma narrativa clara para apresentação ao cliente.

---

## 🧠 Contexto e Abordagem

Com base no relatório já existente, foi criada **nova página** estruturada como se fosse apresentada diretamente a um cliente final. Dessa forma, os visuais foram organizados com foco em **storytelling**, evitando apenas a disposição aleatória de gráficos.

A página conta com **pelo menos dois visuais principais**, controlados por parâmetros, que permitem alternar análises sem duplicar elementos visuais.

---

## 🧩 Diretrizes Aplicadas

### 🔹 Primeira Visão – Parâmetro por Categorias
- Utilização de **parâmetros baseados em categorias** (ex.: Produto, Segmento, País).
- Permite ao usuário alternar dinamicamente a dimensão analisada.
- Facilita comparações e múltiplas leituras dos dados em um único visual.

### 🔹 Segunda Visão – Parâmetro por Valores
- Utilização de **parâmetros baseados em métricas**, como:
  - Sales  
  - Profit  
  - Outras medidas relevantes
- Possibilita analisar diferentes indicadores mantendo o mesmo contexto visual.
